# 快捷键

## chrome

chrome 帮助中，可以找到所有的快捷键

| 操作                     | 快捷键                 |
| ------------------------ | ---------------------- |
| 清除缓存                 | Ctrl + Shift + Delete  |
| 跳转地址栏               | Ctrl + L、Alt + D、 F6 |
| 直接搜索                 | Ctrl + K、Ctrl + E     |
| 重新加载网页（忽略缓存） | Ctrl + Shift + F5      |
| 显示网页的Html代码       | Ctrl + U               |
| 向下滚动网页             | 空格键                 |
| 向上滚动网页             | Shift + 空格键         |
| 打开 Chrome 菜单         | **Alt + e**            |
| 打开“历史记录”页         | **Ctrl + h**           |
| 打开“下载内容”页         | **Ctrl + j**           |
| 打开 Chrome 帮助中心     | **F1**                 |
| 搜索标签页               | Ctrl + Shift + A       |
